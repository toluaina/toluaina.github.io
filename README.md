# toluaina.github.io

Documentation site for [PGSync](https://github.com/toluaina/pgsync), served at [pgsync.com](https://pgsync.com).

Built with [Zensical](https://zensical.org) from `mkdocs.yml` and `docs/`.

## Development

```bash
pip install -r requirements.txt
zensical serve
```

Pushes to `main` deploy automatically to GitHub Pages via `.github/workflows/ci.yml`.
