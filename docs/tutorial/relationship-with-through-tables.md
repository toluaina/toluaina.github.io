Each child node can have only one direct parent.

We can specify the relationship between a parent and child node using the `relationship` property.

```JSON
[
    {
        "database": "book",
        "index": "book",
        "nodes": {
            "table": "book",
            "columns": [
                "isbn",
                "title",
                "description"
            ],
            "children": [
                {
                    "table": "author",
                    "columns": [
                        "id", "name"
                    ],
                    "relationship": {
                        "type": "one_to_many",
                        "variant": "object",
                        "through_tables": [
                            "book_author"
                        ]
                    }
                 }
            ]
        }
    }
]
```
